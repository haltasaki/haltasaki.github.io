\underline{81ページ冒頭から5-4節の最後までを以下で差し替え：}
 \setcounter{footnote}{12}

\bigskip

ここで正の定数 $\alpha$ を
\begin{equation}
\alpha=\frac{\Qmax(T;X_0\to X_1)}{\Qmax(T;Y_0\to Y_1)}
\tag{5.25}
\label{alpha}
\end{equation}
と定める。
$Y$-系をそのまま $\alpha$ 倍にした系を考える。
最大吸熱量の示量性(\ref{Qex})から、この系は \cba{} に対応する $(T;\alpha Y_0)\tooiq(T;\alpha Y_1)$ という操作の間に $\Qmax(T;\alpha Y_0\to\alpha Y_1)=\alpha\Qmax(T;Y_0\to Y_1)$ だけの熱を吸収する。
ところが (\ref{alpha}) より、この吸熱量は $X$-系の (c) での発熱量
 $\Qmax(T;X_0\to X_1)$ に等しい。

%%%%%%the following is new
そこで、$X$-系での (c) の操作 $(T;X_1)\tooiq(T;X_0)$ と $\alpha$ 倍した $Y$-系での \cba{} の操作 $(T;\alpha Y_0)\tooiq(T;\alpha Y_1)$ を同時に行なうことを考える。
そうすると、示量変数の組$(X,\alpha Y)$の複合系についての等温準静操作
\begin{equation}
(T;X_1,\alpha Y_0)\tooiq(T;X_0,\alpha Y_1)
\tag{5.25b}
\label{TXYiqNew}
\end{equation}
が得られる。
最大吸熱量の相加性(\ref{Qadd})と上の段落での評価を使えば、
\begin{align}
\Qmax(T;(X_1,\alpha Y_0)\to(X_0,\alpha Y_1))&=
\Qmax(T;X_1\to X_0)+\Qmax(T;\alpha Y_0\to\alpha Y_1)
\ret&=0
\tag{5.25c}
\label{QmaxTXYNew}
\end{align}
がいえる。{\bf 等温準静操作(\ref{TXYiqNew})を全体としてみれば、系は環境とまったく熱をやりとりしない}のだ。
それなら、(\ref{TXYiqNew})は、等温準静操作であるとともに、{\bf （実質的には）断熱準静操作でもあるとみなしていいだろう} --- というのがCarnotの着想だった。

これは熱力学の本質に迫るきわめて重要なアイディアだが、残念ながら、かなり大雑把な議論でもある。
そもそも最大吸熱量がゼロというのは、操作全体で総計したとき系と環境の間の熱のやりとりがゼロということを意味するに過ぎない。操作の途中では系と環境は熱をやりとりしており、総計としてたまたま吸熱量と発熱量がバランスしているというのが普通だろう\footnote{%
\underline{進んだ注：}仮に操作の途中の任意の時刻で吸熱率と発熱率がバランスするような特殊な状況（本の付録Aでは、そのような状況を構築している）を考えるとしても、（等温環境で行なう）等温操作と（断熱環境で行なう）断熱操作を同一視してよいかは本質的に難しい問題である。
}。

幸いにも、このCarnotの着想を自然に厳密にできることがわかっている（よって、厳密さに深入りしたくない読者はこの着想をそのまま認めて先に進んでいいと思う）。
等温準静操作(\ref{TXYiqNew})そのものを断熱操作とみなすのは無理があるが、実は、始状態も終状態も完全に(\ref{TXYiqNew})と一致する
\begin{equation}
(T;X_1,\alpha Y_0)\tooaq(T;X_0,\alpha Y_1) \; 
\tag{5.26}
\label{CA1}
\end{equation}
という断熱準静操作が存在することが示されるのだ\footnote{%
田中琢真氏の私信（2014年12月）による。
}。
この事実の証明は後の補遺で述べることにして、Carnotの定理の証明を進めよう。
 
ここから先の構成はずっと単純である。
%%%%%%the above is new
(a) の $(T';X_0')\tooiq(T';X_1')$ と \aba{} の $(T';Y_1')\tooiq(T';Y_0')$ を単に組み合わせた等温準静操作
\begin{equation}
(T';X_0',\alpha Y_1')\tooiq(T';X_1',\alpha Y_0')
\tag{5.27}
\label{CI}
\end{equation}
(b) の $(T';X_1')\tooaq(T;X_1)$ と \dba{} の $(T';Y_0')\tooaq(T;Y_0)$ を単に組み合わせた
断熱準静操作
\begin{equation}
(T';X_1',\alpha Y_0')\tooaq(T;X_1,\alpha Y_0)
\tag{5.28}
\label{CA2}
\end{equation}
そして、(d) の $(T;X_0)\tooaq(T';X_0')$ と \bba{} の $(T;Y_1)\tooaq(T';Y_1')$ を単に組み合わせた断熱準静操作
\begin{equation}
(T;X_0,\alpha Y_1)\tooaq(T';X_0',\alpha Y_1')
\tag{5.29}
\label{CA3}
\end{equation}
を用意する。
四つの操作 (\ref{CI}), (\ref{CA2}), (\ref{CA1}), (\ref{CA3}) を続けて行なうことで、サイクル
\begin{equation}
(T';X_0',\alpha Y_1')\tolabel{iq}{a,\abar}
(T';X_1',\alpha Y_0')\tolabel{aq}{b,\dbar}
(T;X_1,\alpha Y_0)\tolabel{aq}{c,\cbar}
(T;X_0,\alpha Y_1)\tolabel{aq}{d,\bbar}
(T';X_0',\alpha Y_1')
\tag{5.30}
\label{Car2}
\end{equation}
が得られる。
%%%%%%the following is new
Carnot サイクル (\ref{Car}) と逆 Carnot サイクル (\ref{CarE}) を連動させ、(c,$\bar{\rm c}$)の部分を断熱準静操作に置き換えたといってもいい。
%%%%%%the above is new
このサイクルは、温度 $T'$ での等温操作 (a,$\rm\bar{a}$) 以外の各操作が断熱操作であり、また全ての操作が準静的なので、温度 $T'$ での等温準静サイクルになっている。
よって、Kelvin の原理~\ref{a:KP} から導かれた結果~\ref{r:Wc=0} により、サイクル (\ref{Car2}) の間に系が外界に行なう仕事 $\Wcyc$ は 0 であることがわかる。

%%%%%%the following is new
同じ仕事$\Wcyc$をサイクル(5.30)の定義から直接求めよう。
この際、等温準静操作(\ref{TXYiqNew})と断熱準静操作(\ref{CA1})において系が外界にする仕事は等しいことに注意する。
これは最大吸熱量がゼロであること(\ref{QmaxTXYNew})と、エネルギー保存則(\ref{W=U-U}), (\ref{Qmax})の帰結である。
よって、サイクル(\ref{Car2})における仕事はもとのCarnotサイクル(\ref{Car})と逆Carnotサイクル(\ref{CarE})における仕事の和になる。
単一のCarnotサイクルの行なう仕事は、(\ref{WCar})のように求められるので、結局、
%%%%%%the above is new
\begin{align}
\Wcyc
=&
\Qmax(T';X_0'\to X_1')-\Qmax(T;X_0\to X_1)
\ret
&
-\alpha\Qmax(T';Y_0'\to Y_1')+\alpha\Qmax(T;Y_0\to Y_1)
\ret
=&
\Qmax(T';X_0'\to X_1')
-\frac{\Qmax(T;X_0\to X_1)}{\Qmax(T;Y_0\to Y_1)}
\Qmax(T';Y_0'\to Y_1')
\tag{5.31}
\label{Wtcyc}
\end{align}
と評価できる。
 $\alpha$ の定義 (\ref{alpha}) を用いた。
これを $\Wcyc=0$ に代入すれば、
\begin{equation}
\frac{\Qmax(T';X_0'\to X_1')}{\Qmax(T;X_0\to X_1)}
=
\frac{\Qmax(T';Y_0'\to Y_1')}{\Qmax(T;Y_0\to Y_1)}
\tag{5.32}
\label{QQES}
\end{equation}
という等式が得られる。
これは、まさに最大吸熱量の比が熱力学的な系と参照点の選び方に依存しないことを示している。
Carnot の定理が示された。


\bigskip
\paragraph*{補遺}
ここでは、等温準静操作(\ref{TXYiqNew})に対応する断熱準静操作(\ref{CA1})が存在することを示す。
そのために、以下の一般的な結果（この結果は、それ自身でも十分に興味深い）を示せばよい。

\bigskip\noindent
{\bf 結果}　任意の熱力学系において等温準静操作
\begin{equation}
(T;X)\tooiq(T;X')
\label{TXiqTXnew}
\end{equation}
が可能であり、また対応する最大吸熱量が
\begin{equation}
\Qmax(T;X\to X')=0
\label{Qmax=0new}
\end{equation}
を満たすとする。
このとき、断熱準静操作
\begin{equation}
(T;X)\tooaq(T;X')
\label{TXaqTXnew}
\end{equation}
が必ず存在する。

\bigskip
等温操作(\ref{TXiqTXnew})に対応する断熱操作が存在することがまったく自明でないのは本文の議論でもわかるだろう。実際、この結果の証明にはKelvinの原理が必要である。

\proof{導出}
結果~\ref{r:comp}（58ページ）より、$(T;X)\tooa(T;X')$または$(T;X')\tooa(T;X)$ という断熱操作の少なくとも一方が実現できる。
仮に前者が実現可能だとして話を進めよう（もし後者が実現可能なら$X$と$X'$を入れ替えればよいので、これで一般性は失われない）。
系を断熱環境におき、示量変数を$X$から$X'$までゆっくりと変化させることで、断熱準静操作
\begin{equation}
(T;X)\tooaq(T';X')
\label{TXaqT'Xnew}
\end{equation}
が実現される。ここで終温度$T'$は未知である。
以下で、実は$T'=T$しかあり得ないことを示す。よって、実現された(\ref{TXaqT'Xnew})が、存在を示したかった断熱準静操作(\ref{TXaqTXnew})に他ならない。

まず$T'>T$と仮定する。
断熱準静操作(\ref{TXaqT'Xnew})の逆操作と存在を仮定した断熱操作$(T;X)\tooa(T;X')$を組み合わせれば、
\begin{equation}
(T';X')\tooaq(T;X)\tooa(T;X')
\label{T'X'aqTXaTX'new}
\end{equation}
という断熱操作が得られる。全体としてみれば、示量変数$X'$を変化させずに温度を下げる断熱操作になっている。
実は、そのような断熱操作は決して実現できないことが簡単に証明できる。
この事実は、本では、Planckの原理（結果6.4, 100ページ）としてまとられている（先の方にあるが、証明に使うのはKelvinの原理だけ）。
こうして矛盾が導かれたので、$T'\le T$と結論される。

次に$T'<T$と仮定する。
もとの等温準静操作(\ref{TXiqTXnew})の逆操作と断熱準静操作(\ref{TXaqT'Xnew})を組み合わせることで、サイクル
\begin{equation}
(T;X')\tooiq(T;X)\tooaq(T';X')\tooig(T;X')
\label{cycleTanaka}
\end{equation}
を作ることができる。
最後の操作では、単に断熱壁を取り除き系を環境と接触させた（このような操作を広義の等温操作という。問題5.2を見よ）。
サイクル(\ref{cycleTanaka})では、系はつねに温度$T$の環境の中にあるので、これは等温サイクルである。
等温準静操作、断熱準静操作それぞれにおけるエネルギー保存則(\ref{Qmax}), (\ref{W=U-U})を用いれば、このサイクルの間に系が外界にする仕事が
\begin{align}
\Wcyc&=U(T;X')-U(T;X)+\Qmax(T;X'\to X)+U(T;X)-U(T';X')
\ret
&=U(T;X')-U(T';X')
\label{WcycTanaka}
\end{align}
であることがわかる。
広義の等温操作で系が仕事をしないこと、(\ref{Qmax=0new})のように最大吸熱量がゼロであることを用いた。
エネルギーが温度の増加関数であることを示す結果~\ref{r:Uinc}（63ページ）より、$T'<T$ならば(\ref{WcycTanaka})の最右辺は正である。
これは$\Wcyc>0$を意味するからKelvinの原理（要請~\ref{a:KP}, 38ページ）と矛盾。
よって$T'\ge T$と結論される。\qed
